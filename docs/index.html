PingWatcher
a tool for monitoring
Installation is easy on MacOS or Linux. Just install the RRD dependency and then install pingwatcher from with python's pip tool.

Install for Linux

# apt install rrd-tool
# pip install pingchecker

Install for MacOS

# brew install rrdtool
# pip install pingchecker
Built in webserver lets you view graphs from whatever host you start on it.
With the ability to send data directly to any statsd server, you can view your data in whatever interface you want.
Simple to use CLI, allows you to start monitoring devices quickly, no configuration files to figure out.
